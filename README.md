📊 XML Data Analysis Project

This is a small practice project focused on parsing XML data and performing basic data analysis using Python.

📌 Project Overview

In this project, I worked with a sample XML dataset containing customer orders. The data was parsed and transformed into a structured DataFrame for analysis.

What I did
Parsed XML data using Python
Converted nested XML structure into a DataFrame
Calculated total revenue per customer
Identified top-selling products based on revenue
Visualized the top 3 products using a bar chart

Tools Used
Python
Pandas
Matplotlib
🎯 Purpose

This project was created as a small hands-on exercise to practice working with XML data, data transformation, and basic visualization techniques.
